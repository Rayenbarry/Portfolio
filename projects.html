Health-Defender - Java Application

Soon a vaccine against covid 19 will be developed and the state needs a system to track vaccine distribution and facilitate patient registration by doctors during vaccination. The application aims to set up a software application that makes it possible to computerize the registration of patients who have taken vaccine and avoid the classic registrations which are currently under forms papers, also it is easy for the State to follow the number of people who have taken the vaccine to better develop the distribution. I used java to develop the application and MySql to develop the database.
